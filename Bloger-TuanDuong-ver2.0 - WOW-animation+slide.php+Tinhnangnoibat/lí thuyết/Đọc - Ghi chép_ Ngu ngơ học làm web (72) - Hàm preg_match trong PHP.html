Phần 72.       Hàm preg_match trong PHP

Đây là clip số 66:

https://www.youtube.com/watch?v=VR5vzzd50pk&index=66&list=PLv6GftO355AsZFXlWLKob6tMsWZa4VCY1

Hàm preg_match và preg_match_all của PHP sử dụng biểu thức chính quy để so khớp và lấy dữ liệu.
Cú pháp hàm preg_match:

int preg_match ( string $pattern , string $subject [, array &$matches [, int $flags = 0 [, int $offset = 0 ]]] )

Hàm preg_match thực hiện so khớp biểu thức chính quy $pattern trên chuỗi $subject. Nếu khớp, trả về giá trị 1; nếu không khớp, trả về giá trị 0; nếu có lỗi trả về giá trị FALSE, giá trị tìm thấy chứa trong mảng $matches.

Ví dụ 1,

$subject = 'Đây là chuỗi ví dụ';
$pattern = '/ví dụ/';
echo preg_match($pattern, $subject);

Ví dụ 2,

$subject = 'Đây là chuỗi ví dụ';
$pattern = '/ví dụ/';
echo preg_match($pattern, $subject, $matches);
echo '<pre>';
print_r($matches);
echo '</pre>';

Hàm preg_match_all trả về kiểu int (số các giá trị tìm thấy), các chuỗi tìm thấy được lưu trong mảng $matches. Ví dụ,

$subject = 'Đây là chuỗi ví dụ để làm ví dụ';
$pattern = '/ví dụ/';
echo preg_match_all($pattern, $subject, $matches);
echo '<pre>';
print_r($matches[0]);
echo '</pre>';

Cặp dấu ## được sử dụng để bao $pattern khi nó có chứa các kí tự đặc biệt. Ví dụ,

$subject = 'Đây là chuỗi /ví dụ/ để làm ví dụ';
  $pattern = '#/ví dụ/#';
  echo preg_match_all($pattern, $subject, $matches);
  echo '<pre>';
  print_r($matches[0]);
  echo '</pre>';

Ví dụ: hàm kiểm tra một email có hợp lệ hay không?

  function checkEmail($value) {
    $pattern = '/^[_a-z0-9-]+(\.[_a-z0-9-]+)*@[a-z0-9-]+(\.[a-z0-9-]+)*(\.[a-z]{2,})$/i';
    $flag = false;
    if (preg_match($pattern, $value) == true) {
      $flag = true;
    }
    return $flag;
  }
  echo checkEmail('nguyenvan.teo@gmail.com');

Biểu thức chính quy, kiểm tra username có hợp lệ hay không (lấy từ stackoverflow):

/^[a-z\d_]{2,20}$/i
||||  | |||     |||
||||  | |||     ||i : case insensitive
||||  | |||     |/ : end of regex
||||  | |||     $ : end of text
||||  | ||{2,20} : repeated 2 to 20 times
||||  | |] : end character group
||||  | _ : underscore
||||  \d : any digit
|||a-z: 'a' through 'z'
||[ : start character group
|^ : beginning of text
/ : regex start

Viết các hàm kiểm tra username, password và website cũng tương tự như email, chỉ thay đổi giá trị của $pattern.

Kiểm tra dữ liệu cho các input của form, nếu dữ liệu bị lỗi thì hiển thị thông báo ngay bên dưới input.

  <?php
  $email = '';
  $username = '';
  $password = '';
  $website = '';
  $errors = array();
  if(!empty($_POST)) {
    $email = $_POST['email'];
    if (checkInput($email, 'email') == false) {
      $errors['email'] = '<p class="error">Địa chỉ email chưa hợp lệ</p>';
    }
    $username = $_POST['username'];
    if (checkInput($username, 'username') == false) {
      $errors['username'] = '<p class="error">Username chưa hợp lệ</p>';
    }
    $password = $_POST['password'];
    if (checkInput($password, 'password') == false) {
      $errors['password'] = '<p class="error">Password chưa hợp lệ</p>';
    }
    $website = $_POST['website'];
    if (checkInput($website, 'website') == false) {
      $errors['website'] = '<p class="error">Địa chỉ website chưa hợp lệ</p>';
    }
  }
  // Hàm kiểm tra cho bốn kiểu dữ liệu nhập vào:
  function checkInput($value, $type='email') {
    switch($type) {
      case 'email':
      $pattern = '/^[_a-z0-9-]+(\.[_a-z0-9-]+)*@[a-z0-9-]+(\.[a-z0-9-]+)*(\.[a-z]{2,})$/i';
      break;
      case 'password':
      $pattern = '/^(?=.*\d)(?=.*[A-Za-z])[0-9A-Za-z!@#$%]{8,12}$/';
      break;
      case 'username':
      $pattern = '/^[a-z\d_]{2,20}$/i';
      break;
      case 'website':
      $pattern = '/^(https?:\/\/)?(www\.)?[-a-zA-Z0-9@:%._\+~#=]{2,256}\.[a-z]{2,6}\b([-a-zA-Z0-9@:%_\+.~#?&\/\/=]*)$/';
      break;
    }
    $flag = preg_match($pattern, $value);
    return $flag;
  }
  ?>

        <?php if (isset($errors['website'])) { echo $errors['website']; } ?>
      </fieldset>
      <input type="submit" value="Gửi">
    </form>
